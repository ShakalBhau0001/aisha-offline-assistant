# 🤖 Aisha Offline Assistant

A **terminal-based smart offline AI assistant** built with Python.  
It works completely offline, remembers your name, responds intelligently, and provides useful features like jokes, quotes, weather, date & time updates 🚀.  
Built with a **beautiful Rich CLI interface** to make chatting fun and interactive 💕  

---

## ✨ Features

- 🧠 Smart offline AI logic (no internet required)
- 💾 Remembers your name during chat session
- 🎨 Beautiful terminal UI (powered by **Rich**)
- 😂 Random jokes
- 💬 Motivation quotes
- 🌦️ Offline weather simulation
- 📅 Current date & time
- 🎨 Dynamic theme color change
- 📂 Chat history auto-save
- ❤️ Beginner-friendly clean code structure

## 🛠️ Installation & Setup

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/ShakalBhau0001/aisha-offline-assistant.git
cd aisha-offline-assistant
```

### 2️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

Or manually:
```bash
pip install rich
```

### 3️⃣ Run the Assistant
```bash
python aisha_bot.py
```

## 🎮 Example Commands

- **hello / hi** → Friendly greeting  
- **my name is Rahul** → Stores your name  
- **what is my name** → Recalls your name  
- **joke** → Random joke  
- **quote** → Motivation quote  
- **weather** → Offline weather info  
- **time** → Current time  
- **date** → Today’s date  
- **color** → Change theme color  
- **help** → Show all commands  
- **bye** → Exit & save chat  

---

## 🧠 How It Works

Aisha runs fully offline using:
- Pattern matching (Regex)
- Keyword-based knowledge system
- Memory dictionary for session storage
- Rich library for UI panels & styling

No API keys.  
No cloud calls.  
Fully private 💻🔒
